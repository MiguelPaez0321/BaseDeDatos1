Performance Analysis of a Convolutional Neural Network for Pneumonia Detection on an Embedded AI System

El artículo analiza cómo se puede implementar una red neuronal convolucional (CNN) para detectar neumonía en sistemas embebidos, utilizando imágenes de rayos X. Evalúan el rendimiento del sistema en función de varios parámetros, como profundidad de color, número de capas, filtros y técnicas de cuantización, considerando aspectos como la precisión, el tiempo de inferencia y el uso de memoria RAM y memoria flash. Los resultados muestran que aumentar ciertos parámetros, como el número de filtros, puede mejorar la precisión, pero también incrementa el consumo de memoria y el tiempo de procesamiento, lo que en algunos casos supera la capacidad del dispositivo embebido. Por ello, resaltan la importancia de ajustar cuidadosamente los hiperparámetros para optimizar la aplicación en sistemas con recursos limitados, especialmente en el contexto de aplicaciones médicas portátiles y en tiempo real.

Más allá de la caja negra: la IA en los tribunales de orden civil (una propuesta heurística)

El artículo explora la posible implementación de modelos de inteligencia artificial basados en redes neuronales supervisadas en la jurisdicción civil española. Propone una metodología experimental para evaluar la fiabilidad, precisión y capacidad explicativa de estos modelos, como los transformadores y BERT, en el contexto judicial. El estudio destaca que, debido a la estructura normativa clara, la disponibilidad de datos históricos etiquetados y la relativamente baja complejidad ética del derecho civil, estos sistemas pueden servir como herramientas complementarias para agilizar procesos, aumentar la coherencia en las resoluciones y ofrecer mayor transparencia, manteniendo siempre el control humano en las decisiones judiciales.

La evaluación del aprendizaje como herramienta de calidad educativa

El documento analiza el proceso de evaluación del aprendizaje, resaltando su papel esencial en la mejora de la calidad educativa. Se revisan las funciones diagnóstica, formativa y sumativa, así como los criterios e instrumentos que permiten valorar de manera justa y objetiva el progreso de los estudiantes. Además, se enfatiza que la evaluación debe ser continua, integral y adaptada al contexto, fomentando la reflexión crítica tanto en docentes como en alumnos.

Web Block Craft: desarrollo web para niños usando Google Blockly

El artículo presenta Web Block Craft, una aplicación educativa innovadora que utiliza el framework de Google Blockly para enseñar desarrollo web a niños y principiantes. La plataforma ofrece un entorno de aprendizaje integral que combina proyectos frontend (HTML, CSS, JavaScript, manipulación del DOM) y backend (servidores, seguridad web, APIs y bases de datos). Gracias a su interfaz de bloques arrastrables y visualización en tiempo real, los estudiantes pueden comprender la relación entre los bloques y el código generado. Además, incorpora funciones como lecciones paso a paso para docentes, documentación extensa y herramientas de depuración, lo que convierte la experiencia en un proceso dinámico, accesible y atractivo. El estudio concluye que la aplicación facilita la comprensión de conceptos complejos de programación web y fomenta la creatividad y el interés de los jóvenes por el desarrollo tecnológico.

Una implementación de problemas de relleno de elementos en blanco para el estudio de la comprensión del código de programación de clientes web basada en JavaScript

El artículo propone la implementación de problemas de relleno de elementos en blanco (EFP) en el sistema de autoaprendizaje de programación (PLAS), orientado a mejorar la comprensión del código en la programación de clientes web con HTML, CSS y JavaScript. En esta metodología, se presentan fragmentos de código con espacios vacíos que los estudiantes deben completar correctamente, apoyándose en capturas de pantalla de las páginas web resultantes. Se generaron 21 instancias de EFP, probadas con 20 estudiantes de maestría en la Universidad de Okayama, quienes no tenían formación previa en programación web. Los resultados mostraron una alta tasa de respuestas correctas (más del 96%), lo que confirma la eficacia de esta técnica para facilitar el aprendizaje de la lógica y la integración de etiquetas, estilos y funciones en JavaScript. El estudio concluye que los EFP constituyen una herramienta efectiva para el autoaprendizaje y comprensión del desarrollo web del lado del cliente.

Evaluación del rendimiento del web scraping mediante XPath, selector CSS, expresiones regulares y HTML DOM con aplicaciones técnicas de multiprocesamiento

El artículo presenta un estudio comparativo sobre el rendimiento de diferentes métodos de web scraping, incluyendo XPath, selectores CSS, expresiones regulares y HTML DOM, con el objetivo de identificar cuál resulta más eficiente en distintos escenarios. Se analizan métricas como el tiempo de ejecución, uso de memoria y precisión en la extracción de datos. Además, se evalúa el impacto de aplicar técnicas de multiprocesamiento para mejorar la velocidad de procesamiento en grandes volúmenes de información. Los resultados muestran que XPath y selectores CSS ofrecen un mejor balance entre precisión y eficiencia, mientras que las expresiones regulares son útiles en casos específicos pero menos robustas. El multiprocesamiento mejora significativamente el rendimiento global, reduciendo los tiempos de ejecución en tareas intensivas de scraping. El estudio concluye que la elección de la técnica debe adaptarse al tipo de datos, la complejidad del sitio web y los recursos disponibles.
